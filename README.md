Features
---
Some poison features, and what will be added:
- [x] Random user-agent
- [x] Wordlist with 80K files, directories, and more
- [x] Multi-threading
- [ ] Custom cookies
- [ ] Wordlist categories
- [ ] Proxy or random proxy

Install
---

Download poison by cloning the Git:
```
git clone https://github.com/yexploit/poison.git/ poison
```
Running:
```
cd poison
ruby poison.rb -h
```
